Pong Game - Luke Monaghan

Controls.
	Menu
		Up and Down arrows to move the cursor, enter key to go between menu's.
	Gameplay
		Left paddle is moved with the W and S keys, W is up, S is down.
		Right paddle is moved with the Up and Down arrow keys respectivly.
		Backspace to show debug information, Text outputs to the console window.
Game Dynamics.
	The game is played like normal pong, paddles move on the up and down Y axis.
	The ball is moved when hit, the paddle does move the ball depending on where it is hit (3 areas per paddle)
	First to score 10 points on the other player wins.
